🎓 MineAI Tutor

An AI-powered tutoring assistant built for Italian students, created by a 14-year-old developer from Italy.

Live Demo: mineai-teach.streamlit.app

🤖 What is MineAI?
MineAI is a web-based AI assistant designed to help students with homework, research, and learning. It combines multiple AI models with real-time web search to provide accurate, up-to-date answers.

⚠️ Note: The AI interface and responses are in Italian, as MineAI is built for Italian students.


✨ Features

Multi-model AI — Llama 3.1 8B (Standard) and Llama 3.3 70B (PRO) via Groq
Real-time web search — Active on every query for up-to-date answers
Image analysis — Upload a photo of an exercise and MineAI will solve it
Focus modes — General, Technical Analysis, Copywriting, Coding
Business Intelligence dashboard — Upload CSV/XLSX and visualize data interactively
Persistent chat history — Conversations saved across sessions
Word report export — Download AI responses as .docx files (PRO users)
User authentication — Secure login and registration system
