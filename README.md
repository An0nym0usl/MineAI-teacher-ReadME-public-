# 🎓 MineAI Tutor

An AI-powered tutoring assistant built for students, created by a 14-year-old developer from Italy.

**Live Demo:** [mineai-teach.streamlit.app](https://mineai-teach.streamlit.app)

## 🤖 What is MineAI?

MineAI is a web-based AI assistant designed to help students with homework, research, and learning.  
It combines multiple AI models with real-time web search to provide accurate, up-to-date answers.

> ⚠️ **Note:** The interface is currently available in Italian and English.  
> AI responses generally adapt to the user's language, but there may still be minor bugs.

## ✨ Features

- Multi-model AI: Llama 3.1 8B (Standard) and Llama 3.3 70B (PRO) via Groq
- Real-time web search: active on queries to provide up-to-date information
- Image analysis: upload a photo of an exercise and MineAI can analyze/solve it
- Focus modes: General, Technical Analysis, Copywriting, Coding
- Business Intelligence dashboard: upload CSV/XLSX and visualize data interactively
- Persistent chat history: conversations saved across sessions
- Word report export: download AI responses as `.docx` files (PRO users)
- User authentication: login and registration system with Supabase

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Frontend | Streamlit |

| Vision | Llama vision model via Groq |
